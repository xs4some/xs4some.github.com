New Theme

by Hendrik Bruinsma - Apr 6th, 2014 -

Some time ago, I started using Octopress to generate my website. Now I finally to the time the install a theme, so it looks less generic.

One thing I pertically like about Octopress, is the use of Markdown to write posts and pages. It really beats HTML, as only the content is part of your post instead of being little collections of HTML code.
